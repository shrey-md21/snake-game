# Snake Game

## Overview

Welcome to the classic old Snake Game! The most simple yet addictive game involves controlling a snake as it moves around the screen, eating apples to grow longer while avoiding the boundaries of the screen and its own body.

## How to Play

1. **Controls**: Use the arrow keys to control the direction of the snake. 

2. **Objective**: Guide the snake to eat as many apples as possible to increase your score. Each time the snake eats an apple, it grows longer.

3. **Game Over**: The game ends if:
    - The snake collides with the edges of the screen.
    - The snake collides with its own body.

## Scoring

- Each apple eaten increases the player's score.
- The score is a reflection of how many apples the snake has consumed.

## Features

- **Dynamic Growth**: As the snake eats apples, it grows longer, increasing the challenge of navigating the screen without colliding with obstacles.
- **Score Display**: The current score is displayed on the screen, allowing players to track their progress.
- **Endless Gameplay**: The game continues until the snake collides with a wall or itself, providing players with the opportunity to achieve higher scores with each attempt.

## Getting Started

To start playing the Snake Game, download or clone the game files and open the main executable file.
